# Infographic

Use this skill when the user asks for an infographic, visual summary, information card, comparison visual, roadmap, matrix, or high-density explainer.

Default output is a structured design brief or a self-contained HTML/SVG artifact. Use image generation only when explicitly requested and available.

## Layout Choices

- Timeline or linear progression for processes.
- Comparison matrix for alternatives.
- Hub and spoke for a central concept with related ideas.
- Bento grid for mixed facts and highlights.
- Dashboard for metrics and KPIs.
- Roadmap for milestones.

## Workflow

1. Reduce the source material to 5-9 major points.
2. Pick one layout and one visual tone.
3. Draft the hierarchy: title, sections, captions, and supporting facts.
4. Produce the artifact or prompt set.
5. Verify that text remains legible and no labels overlap.

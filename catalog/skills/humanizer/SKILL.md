# Humanizer

Use this skill when the user asks to make writing sound more natural, less AI-like, warmer, clearer, or more conversational.

## Workflow

1. Preserve meaning, factual claims, names, numbers, and constraints.
2. Remove filler, generic transitions, over-explaining, and unnatural symmetry.
3. Vary sentence length while keeping the user's intended tone.
4. Prefer concrete verbs and plain phrasing.
5. Return the revised text, and only explain edits when asked.

## Avoid

- Do not add facts.
- Do not make professional writing overly casual.
- Do not imitate a living person's private style.
- Do not hide policy, legal, medical, or financial caveats that matter.

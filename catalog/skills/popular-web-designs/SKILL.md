# Popular Web Designs

Use this skill when the user asks for a page, prototype, or UI that should feel like a known product style such as Stripe, Linear, Vercel, Notion, or Airbnb.

This catalog package provides routing guidance, not a full template library. Use public visual patterns as inspiration without copying protected assets, logos, or proprietary source.

## Workflow

1. Identify the named reference style and the user's actual product domain.
2. Translate the reference into general tokens: spacing, density, typography, color role, motion, and component feel.
3. Build original UI content for the user's task.
4. Avoid exact cloning of brand pages, logos, or trademarked compositions.
5. Verify responsive behavior and text fit.

## Pair With

- `claude-design` for one-off artifact process and taste.
- `design-md` when the user wants durable token documentation.

# Global skills available from trusted AHA core

This generated index helps agents discover trusted harness skills quickly.
It is a reading surface, not a runtime, and it does not auto-install anything.

## How to use

1. Match the task to a skill name or description.
2. Read the selected `SKILL.md` first.
3. Ask the user before installing any optional catalog skill.
4. Use `aha skills search` when you need catalog discovery beyond the core set.

## Discovery Hints

- `aha bridge` points to the correct user skill or harness skill surface.
- `skills search` helps you find optional catalog entries after user approval.
- `skills install` must not auto-install; ask the user first.
- `ask the user` is the default when a catalog skill is only a recommendation.
- `do not auto-install` is a hard boundary for optional skills.
- `when to recommend` is determined by the task, not by memory alone.
- `already installed` means use the installed skill rather than reinstalling it.
- `frontend-design` and `xlsx` are examples of optional catalog skills that may be relevant outside the harness core.

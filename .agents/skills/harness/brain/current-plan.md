# Current Execution SSOT

Current execution SSOT is not stored in this brain directory.

Use these files instead:

- `.agents/mission/plan.json` - machine-readable active/archive execution plan registry.
- `.agentos/project/exec-plans/active/` - reviewed plans that may be executed after the required gates.
- `.agentos/project/exec-plans/archive/` - completed, rejected, or superseded plans.
- `.agents/traces/harness/loop-state.md` - loop-mode runtime state only.
- `HISTORY.md` - durable checkpoints, approvals, loop stops, and verification evidence.

This file is only a pointer so stale task boards do not become operational instructions.

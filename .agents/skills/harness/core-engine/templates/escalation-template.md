# Escalation Template

> Rule 1 또는 Rule 2 발동 시 이 형식을 반드시 사용할 것.

```
## [ESCALATION] 에이전트 판단 요청

### 상황
- **작업명**: [무엇을 하려고 하는가]
- **중요도**: [고/중/저]
- **진행 상태**: [어디까지 진행했는가]
- **불확실성 유형**: [데이터 부족 | 모순 | 환경 변화 | 반복 오류]
- **신뢰도 갭**: [현재 신뢰도] vs [필요 신뢰도]

### 선택지
**A**: [설명] — 장점: [...] / 단점: [...]
**B**: [설명] — 장점: [...] / 단점: [...]
**권장**: [A 또는 B] — 이유: [근거]

### 요청
위 선택지 중 어떤 것을 선택하시겠습니까?
```
